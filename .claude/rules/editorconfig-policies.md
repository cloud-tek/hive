# editorconfig-policies.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository, in regards of editor coding rules and code styling

## Critical Rule: Enforcement of .editorconfig

When generating code for this repository, Claude Code MUST strictly adhere to the coding style rules defined in the repository's .editorconfig file. This includes, but is not limited to:
- Indentation style and size
- Line endings
- Character encoding
- Trailing whitespace
- Final newline at end of files
